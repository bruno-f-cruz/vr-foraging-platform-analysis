# Getting Started

1. Install uv (https://docs.astral.sh/uv/getting-started/installation/)
2. run `uv sync` to synchronize dependencies and create a `.venv`
3. (Optional) run `uv run download.py` to download all data from aws
4. Open the `demo.ipynb` notebook in Jupyter Notebook or JupyterLab
5. Select the local `.venv` kernel in Jupyter Notebook or JupyterLab
6. Run notebook
